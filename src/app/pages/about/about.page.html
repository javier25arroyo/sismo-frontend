Acerca de la aplicación

Esta aplicación sirve como base para un frontend en Angular con autenticación, navegación protegida y componentes reutilizables.

La refactorización actual reemplaza estilos CSS por utilidades de Tailwind para simplificar mantenimiento, consistencia visual y velocidad de iteración.

Stack actual

  • Angular standalone components
  • Tailwind CSS para layout y UI
  • Rutas públicas y protegidas
  • Servicios para API y autenticación
